minimalist, flat nix environment. pure and reproducible, for you to modify and build with.

commands (local):

curl -L https://github.com/estradapabloedrei/rebranded-modular/archive/main.tar.gz | tar -xz --strip-components=1

- aims to be 100% of the following :
reproducible
deterministic
declarative
eternal
atomic

- tries to be 0% of the following :
unopinionated
vegan
